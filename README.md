## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

### Prerequisites
```bash
# nodejs 18
# pnpm installed
```

## Installation

```bash
$ pnpm install
```

### Devlopment
```bash
# postgres
$ docker-compose --env-file ./.env.dev up -d
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
